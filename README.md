# task-mini-framework

## Description

### build using webpack

```bash
npx webpack --config webpack.config.js
```

### build using rollup

```bash
npx rollup -c rollup.config.mjs
```

### run
- open `index.html` in browser  
or
- run live server in vscode  
or
- `npm start`  
or
- `npm run start`
